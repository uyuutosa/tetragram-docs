# pentaglyph-docs two-axis taxonomy matrix — methodology spec for paperbanana

## トーン & マナー（必読）

- 16:9 横長（3584x2048）。`STRATEGY.md §3.3` の Combined Matrix を視覚化する。`README.md` の "Two-axis taxonomy" 解説セクションに埋め込む。
- 背景: 非常に薄いオフホワイト #F7F8FA のフラットカラー（テクスチャ・ノイズ・写真質感は厳禁）。
- スタイル: モダンでテクニカル、calm。Accenture スライド風の上品な余白感、ただし軽やか。`hero.png` と `layers.png` のデザイン言語に厳格に従う（同じシリーズの 3 枚目という統一感を出す）。
- 立体表現の禁止: 影・グロー・グラデーション・3D・スキュアモーフィズム・写真テクスチャはすべて禁止。線と面のみで構成する。
- 顔・手・人物・絵文字・アイコン風イラストは一切入れない。

## なぜこの図が必要か（design intent、生成自体には使わないが守る目的）

`layers.png` は concern 軸 6 層（⓪/①/②/③/④/⑤）のみを縦に積んだ図で、change-rate 軸（A/B/C）が描かれていない。読者（特に AI agent）は `layers.png` だけ見ると「pentaglyph は 1 軸構造だ」と誤読する。本図は **3 列（change-rate: A/B/C） × 6 行（concern: ⓪/①/②/③/④/⑤）の matrix** を提示し、空セル（intentionally empty）も含めて 2 軸構造であることを一目で理解させる。

## 全体レイアウト（厳守）

横 3584 × 縦 2048 の canvas を **3 つの帯**に分割する:

- **TOP BAND**（上 12% / y=0..245）: タイトル領域
- **MATRIX BAND**（中 80% / y=245..1885）: matrix 本体（列ヘッダー + 6 行 × 3 列のセル）
- **BOTTOM BAND**（下 8% / y=1885..2048）: 下キャプション領域

## TOP BAND — タイトル

中央寄せで 2 行配置。

- 1 行目（メインタイトル）:
  - テキスト: `Two-axis taxonomy`
  - サイズ: 約 120px
  - フォントウェイト: bold
  - 字間: 少し詰める
  - カラー: ダークスレート #111827
- 2 行目（サブタイトル）:
  - テキスト: `Change-rate × Concern — every artefact lives in exactly one cell.`
  - サイズ: 約 42px
  - フォントウェイト: regular
  - カラー: #4B5563（やや薄いグレー）
- 1 行目と 2 行目の間隔: 約 20px
- タイトル全体は TOP BAND の中で垂直中央に置く

## MATRIX BAND — 6 行 × 3 列の matrix 本体

MATRIX BAND を **左 6%（x=0..215）の "concern axis ガター"** + **残り 94%（x=215..3584）の matrix 領域** に分割する。

### concern ガター（左端、x=0..215）

ガターの中に **6 行ぶんの concern ラベル**を縦に並べる。各 concern ラベルは MATRIX BAND の上下を 6 等分した各バンドに垂直中央で置く。

各 concern ラベルの構造（上から下へ）:

1. **円形番号バッジ**: 直径 96px、塗りつぶしカラーは concern のアクセントカラー、内側に層番号テキスト（⓪/①/②/③/④/⑤、サイズ約 52px、bold、白文字、水平・垂直中央）。⑤ は背景 #4B5563 のまま 50% 透明
2. **層名テキスト**（バッジの右、ガター幅 215px の中に収める形にするか、バッジの直下に縦書きで重ねるか）: 採用は **バッジ直下に 1 行**、サイズ 22px、weight medium、カラー #111827、英語名のみ。日本語名はガター幅の都合で省略
   - ⓪: `Standards`
   - ①: `Artefacts`
   - ②: `Process`
   - ③: `Automation`
   - ④: `Governance`
   - ⑤: `Measurement` （文字色 #6B7280 のやや薄色、optional 表現）

### matrix 領域（x=215..3584 = 幅 3369px、y=245..1885 = 高さ 1640px）

matrix を **3 列ヘッダー（高さ 約 180px）** + **6 行 × 3 列のセル領域** に分ける。

#### 列ヘッダー帯（matrix 領域の上 180px）

3 列を等幅に分割（各列 約 1123px 幅）。各列の中央に **ピル型ヘッダー**を配置する。

- ピルの寸法: 幅 約 1000px × 高さ 約 100px、border-radius 約 50px
- ピル内のテキスト: 上記の列ヘッダー文字（サイズ 約 36px、weight bold、水平・垂直中央）
- ピル直下の中央: サブテキスト（サイズ 約 22px、weight regular、カラー #6B7280、ピルの下に 20px の隙間で置く）

#### セル領域（matrix 領域の残り、6 行 × 3 列）

セル領域の高さ約 1460px を 6 行に等分（各行 約 243px）。各行の幅 3369px を 3 列に等分（各列 約 1123px）。

各セルの基本構造:

- セル枠: 1px ヘアライン、カラー #E5E7EB（matrix 全体に格子状に引く）
- セル背景: 純白 #FFFFFF（**ただし intentionally-empty セルは別扱い**、後述）
- セル内部の上端から 20px 下げた位置に、セル左寄せ（セル左端から 32px インデント）で本文テキストを配置

各セルに入れるテキストは、下記「セル本文一覧」を厳守。

## 否定制約（守らないと不合格）

- 絵文字を一切使わない（U+1F300〜U+1FAFF などの emoji codepoint はすべて禁止）。**ただし concern 番号 ⓪①②③④⑤ は Unicode CIRCLED DIGIT (U+24EA, U+2460-U+2464) なので使用可**
- 影・グロー・グラデーション・透視図・3D 表現はすべて禁止
- 写真・写真合成・テクスチャ素材は使わない
- 顔・手・人物のシルエットは入れない
- concern アクセント 6 色（#1F6FEB / #0B8A3A / #B45309 / #6B46C1 / #BE185D / #4B5563）は **左端ガターの番号バッジと、その直下の英語名テキストの強調にのみ**使う。matrix セル内・列ヘッダー・タイトル・キャプション・背景には絶対にブリードさせない
- 列ヘッダーピルの色は 3 段階の灰色（#111827 / #6B7280 / #D1D5DB）のみ。アクセントカラーを混ぜない
- matrix は **6 行 × 3 列の格子**。行を増やしたり減らしたり、列を 4 列にしたり、セルを merge したりしない
- 空セルは "—"（U+2014, EM DASH 1 文字）+ `intentionally empty` の italic 注記のみ。"N/A" / "—（理由）" / "✕" / "空欄" 等の表記は使わない
- ⑤ Measurement 行の "optional" 表現はカード半透明 + やや薄色文字のみ。バッジを破線にしたり、行全体を別枠で囲ったりしない（matrix の格子構造を維持する）
- ロゴ、QR コード、URL、補足の説明文ボックス、`hero.png` / `layers.png` の他のテキストは追加しない
- 番号バッジは **円形のみ**。角丸長方形・六角形・五角形・盾型・リボン型は禁止
- セル内の本文テキストは **等幅フォント**（モノスペース、例: SF Mono / JetBrains Mono / IBM Plex Mono のいずれか）。プロポーショナル体は使わない（パス文字列を読みやすく揃えるため）
- matrix の格子線は #E5E7EB の 1px ヘアラインのみ。太線・破線・色付き線は使わない

## 検証チェックリスト（生成後の VLM 自動レビューでこの順に検査）

1. タイトル「Two-axis taxonomy」が canvas 上部にあり、サブタイトル「Change-rate × Concern — every artefact lives in exactly one cell.」が直下にある
2. canvas の左端に **縦に 6 つの円形番号バッジ** ⓪/①/②/③/④/⑤ が上から下に並んでいる
3. 各バッジの背景色が指定どおり: ⓪青 / ①緑 / ②琥珀 / ③紫 / ④薔薇 / ⑤灰（半透明）

7. 各セルが #E5E7EB のヘアラインで区切られている
