# Roadmap — `<client-name>`

> Authored using the **Now / Next / Later roadmap** pattern (Janna Bastow / ProdPad — <https://productmanagementresources.com/now-next-later-roadmap/>). Three rolling buckets, single living file, edited in place. **No Gantt charts**. The forcing function is that "Now" must be actually-being-worked-on, "Next" must be ready-to-pick-up, and "Later" must be plausibly-relevant — anything that doesn't meet those bars gets parked or dropped.

> **Review cadence**: monthly, owned by the engagement lead with client sponsor input. See [`OPERATING-AGREEMENT.md` §1](./OPERATING-AGREEMENT.md).

> **Companion artefacts**: items in *Later* that need exploration become [`prfaqs/`](./prfaqs/) entries. Items in *Now* are tracked in [`01-artefacts/reports/YYMMDD/`](./01-artefacts/reports/) progress sections. Decisions about which bucket an item belongs in are captured in [`daci/`](./daci/) → [`decisions/`](./decisions/).
